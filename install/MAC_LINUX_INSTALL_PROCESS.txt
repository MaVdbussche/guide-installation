Procédure d'installation Linux pour Mac avec Processeur Intel et mono disque dur : 

-Libérer autant d'espace disque que néccessaire avec le gestionnaire de disque Apple et le mettre en "Espace libre" ou "Free Space”. Si ce n’est pas possible, formater en FAT32 (MS-DOS Fat) afin d’éviter de se tromper par après.

-Telecharger l'EFI pour pouvoir choisir l'OS au démarrage sans l'installer:
	*Pour les versions antérieurs à Yosemite : 
	 -http://refit.sourceforge.net/

	*Pour les version supérieurs ou égale à Yosémite: 
	 -Télécharger rEFInd : http://sourceforge.net/projects/refind/

-Décompresser le .zip. Le laisser dans le dossier Downloads.

-Redémarrer en Recovery mode (appuyer sur pomme-R au moment ou il s’allume.)

-Lancer le terminal (Utilities > Terminal), se rendre dans le dossier décompressé.
-Taper “csrutils disable”
-Taper “./refind-install”
-Taper “csrutils enable”

-Ensuite on peut booter sur une clef usb. (Il faut peut-être appuyer sur pomme ou alt au démarrage.

-Lancer l'installation de linux (le type de clavier pour les macbooks est : French Macintosh)

-Suivre la procédure habituel d'installation jusqu'au choix d'installation
	-Choisir "Installation personnalisée" ou "Autre chose" (!!! surtout pas une autre option !!!)
		*Supprimer la partition FAT16 et y mettre root et un peu de swap.
		*Choisir comme "Boot Loader" la partiton root (en général "/dev/sda4") et surtout pas la racine du disque dur comme sur une installation avec windows ("/dev/sda").
		*Créer une partion de type BIOS ou BIOS-grub de minimum 1Mb
	-Lancer l'installation
	-Terminer l'installation de façon classique
	-Retirer la clé USB

-Redémarrer
	*Lorsque l'écran s'allume, appuyer sur ALT pour afficher les différents partitions bootables, 
		-->normalement, MacOS (Macintosh HD) et Linux sont présent.
	*Tester d'abord Linux
	*Tester ensuite que MacOS est toujours fonctionnel
